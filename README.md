# Leobog Keyboard Audio Visualizer 🎹🎵

Аудио-визуализатор реального времени для клавиатур с поддержкой **OpenRGB** (проверено на Leobog Hi75c Pro, но подходит для любой клавиатуры, совместимой с OpenRGB).

Программа захватывает системный звук (loopback) и выводит динамические световые эффекты на клавиатуру, оптимизируя отправку пакетов для защиты USB-шины от перегрузки.

---

## ✨ Особенности

*   **4 режима работы**:
    *   `spectrum_linear` — частотный спектр распределяется слева направо.
    *   `reactive_pulse` — вся клавиатура пульсирует в такт громкости и меняет цвета.
    *   `hybrid_equalizer` — поклавишный эквалайзер поверх динамической фоновой подсветки.
    *   `standalone_pattern` — работа в режиме генератора эффектов без использования звука.
*   **Умное распознавание битов**: при сильном ударе баса направление движения градиента меняется на противоположное, создавая реактивный эффект.
*   **Спектральный поток (Spectral Flux)**: скорость переливания и насыщенность цветов подстраиваются под темп и агрессивность музыки.
*   **Плавное сглаживание**: раздельное затухание для басов (быстрое) и высоких частот (плавное) предотвращает мерцание диодов.
*   **Авто-переключение при тишине**: если музыка не играет дольше 5 секунд, клавиатура автоматически переходит в мягкий фоновый режим (Matrix, Breath, Wave или Fire) и возвращается к эквалайзеру при появлении звука.
*   **Поддержка 2D-матрицы**: эффекты дождя (Matrix) и пламени (Fire) адаптируются под физическую сетку вашей клавиатуры, если она поддерживается в OpenRGB.

## 🛠 Требования и установка

### Шаг 1: Настройка OpenRGB
1. Скачайте и запустите [OpenRGB](https://openrgb.org/).
2. Перейдите во вкладку **SDK Server** и нажмите **Start Server** (порт по умолчанию: `6742` или `6789`).
3. Убедитесь, что ваша клавиатура определилась в списке устройств.

### Шаг 2: Установка Python и зависимостей
1. Установите **Python 3.8+**.
2. Установите библиотеки проекта с помощью `requirements.txt`:
   ```bash
   pip install -r requirements.txt
   ```

---

## 🚀 Запуск и конфигурация

Запустите скрипт:
```bash
python keyboard_visualizer.py
```

### Настройка эффектов
Вы можете открыть [keyboard_visualizer.py](keyboard_visualizer.py) в любом текстовом редакторе и настроить переменные в блоке **КОНСТАНТЫ НАСТРОЙКИ**:

*   `MODE` — выберите режим работы (`'spectrum_linear'`, `'reactive_pulse'`, `'hybrid_equalizer'`, `'standalone_pattern'`).
*   `PALETTE` — выберите цветовую схему (`'cyberpunk'`, `'sunset'`, `'ocean'`, `'matrix'`, `'rainbow'`).
*   `PATTERN` — эффект при тишине (`'wave'`, `'breath'`, `'matrix'`, `'fire'`, `'none'`).
*   `FPS` — частота обновления кадров (по умолчанию `45`).
*   `LIGHT_DIFFUSION` — размытие цвета между соседними клавишами (для более плавного перелива).

---

## ⚠️ Устранение неполадок

*   **Не подключается к OpenRGB**:
    Убедитесь, что сервер SDK запущен в приложении OpenRGB. Скрипт автоматически пробует порты `6789` и `6742`.
*   **Не захватывает звук**:
    Скрипт пытается автоматически найти системное Loopback-устройство аудиовыхода. Убедитесь, что в системе выбран правильный источник звука по умолчанию (динамики/наушники).
